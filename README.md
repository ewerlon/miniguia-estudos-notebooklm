## 🧠 Engenharia de Prompts e "Cicatrizes"

Nesta seção estão documentadas as estratégias utilizadas para extrair o melhor da IA.

### 🔍 Perguntas Estratégicas

* "Explique [conceito] de forma simples e com exemplos práticos."
* "Quais são os principais pontos de divergência entre [teoria A] e [teoria B]?"
* "Resuma este conteúdo em tópicos estruturados."

### 🔁 Variações de Prompts

| Objetivo          | Prompt utilizado                               |
| ----------------- | ---------------------------------------------- |
| Simplificação     | "Explique como se eu fosse iniciante..."       |
| Aprofundamento    | "Aprofunde tecnicamente incluindo detalhes..." |
| Aplicação prática | "Dê exemplos reais de uso..."                  |

### ⚠️ Dificuldades e Troubleshooting

Durante o processo, algumas dificuldades foram encontradas:

* Respostas genéricas → resolvido com prompts mais específicos
* Falta de contexto → resolvido incluindo trechos das fontes
* Informações imprecisas → conferência cruzada com fontes

> 💡 Aprendizado: A qualidade da resposta está diretamente ligada à qualidade do prompt.
